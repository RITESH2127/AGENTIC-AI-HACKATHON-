## 🌟 Overview

Welcome to the **Agentic AI Hackathon** project. This repository showcases a sophisticated multi-agent system designed to handle complex workflows—from scouting information to quantitative GitHub analysis. By leveraging a modular architecture of specialized agents, this system transforms raw data into actionable insights through an intelligent API gateway.

### Why this project?
Traditional automation is linear. Our **Agentic AI** approach is non-linear and collaborative. Each agent is an expert in its domain, working in harmony to solve tasks that require more than just a simple script.

---

## 🛠️ The Agent Squad

The system is powered by three specialized AI entities:

| Agent | Name | Responsibility |
| :--- | :--- | :--- |
| 🕵️‍♂️ | **Agent Scout** | Performs deep-dive research and information gathering across various sources. |
| 📊 | **GitHub Quant** | Analyzes repositories, evaluates code quality, and provides quantitative metrics. |
| 🤝 | **Partner Agent** | Facilitates collaboration and integrates third-party partnership data. |

---

## 🚀 Key Features

*   **Asynchronous Orchestration:** Built with `aiosqlite` and `FastAPI` for high-performance, non-blocking operations.
*   **LLM Integration:** Powered by **Groq** and **LangChain** for lightning-fast inference and complex reasoning chains.
*   **Unified API Gateway:** A single entry point (`api_gateway.py`) to manage and route requests to the appropriate specialized agents.
*   **Live Web Dashboard:** Includes a sleek `index.html` frontend to interact with the agents in real-time.
*   **Modular Design:** Easily swap or add new agents to the ecosystem without breaking the core logic.
